Project Overview

ASCII Art Web is a project that consists in creating a web server from which it is possible to use a web graphical user interface

Project Features

  1. Accepts user's input from the textarea
  2. Supports choosing from various banner files for output
  3. Supports switching across various theme modes
  4. Responsive web design

Usage Instructions

  1. Ensure you are in the project's home page. You can switch to the home page by clicking on the Home navigation link.
  2. Enter the text you wish to display using ASCII characters in the textarea section
  3. Select your preferred banner file from the select tag
  4. Click on the submit button to generate the ASCII characters
